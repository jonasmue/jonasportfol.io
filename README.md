# jonasportfol.io
Personal Portfolio 👨🏻‍💻

Accessible here: [jonasportfol.io](https://jonasportfol.io)

# Instructions

## Installation
```
git clone https://github.com/jonasmue/jonasportfol.io.git
cd src
npm install
```

## Start Development Server
```
npm run dev
```

## Deploy
```
npm run build
npm run start
```
